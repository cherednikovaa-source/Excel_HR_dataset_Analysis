# Excel_HR_dataset_Analysis

# Project Overview

This project analyzes HR data to identify key patterns related to employee turnover, salary distribution, and career development characteristics.
The goal is to derive practical insights that can support HR and management decision-making.

# Data
- Source: Kaggle (HR dataset)
- Format: CSV / XLSX

# Objectives

- Understand employee turnover patterns across departments and demographic groups
- Analyze salary distribution by department, gender, age, and experience
- Identify potential factors affecting employee retention and compensation
- Provide data-driven insights and recommendations for HR decision-making

# What I did in the project
- Cleaned and validated raw HR data using Power Query
- Performed salary and turnover analysis across multiple dimensions
- Identified patterns related to dismissals and employee experience
- Built an interactive Excel dashboard with charts and slicers

# Tools & Skills

1. Excel
- PivotTables
- PivotCharts
- Power Query (data cleaning and transformation)
- Slicers
- Data visualization (dashboard)

2. Data Analysis Skills
- Data cleaning & validation
- Exploratory data analysis (EDA)
- Insight generation
- Business-oriented interpretation

# Key Insights
- Employee turnover is highest among newcomers, while experienced employees demonstrate strong retention.
- Salaries vary significantly by department, indicating department-level compensation structures.
- There is no significant salary difference between men and women.
- Salary levels do not increase proportionally with age or working experience, suggesting flat salary progression over time.

# Recommendations
- Review onboarding and early-career support processes to reduce turnover among new employees.
- Evaluate career development and compensation progression frameworks to improve long-term motivation.

# Dashboard
A consolidated Excel dashboard was created to provide an interactive overview of key metrics:
- Employee turnover
- Salary distribution
- Demographic breakdowns
The dashboard includes slicers for flexible data exploration.

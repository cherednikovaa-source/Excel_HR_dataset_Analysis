# Excel_HR_dataset_Analysis

This project is an analysis of open HR data from Kaggle in Excel. It includes an Analysis of salaries, turnover, and career development of employees by departments.

## Data
- Source: Kaggle (HR dataset)
- Format: CSV / XLSX

## What I do in the project
- Data cleaning
- Analysis of salaries, turnover by departments
- Finding dismissal/performance patterns
- Building a dashboard in Excel

## Tools
- Excel
- GitHub
